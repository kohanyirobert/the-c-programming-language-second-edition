= About

Solutions for exercises in _The C Programming Language, Second Edition_.

== Requirements

* https://cmake.org[CMake] (3.12.x)
* https://www.python.org[Python] (3.x)

== Building

```
mkdir build
cd build
cmake -G 'Unix Makefiles' ..
make
ctest
```
